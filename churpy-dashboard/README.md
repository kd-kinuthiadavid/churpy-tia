## How To Run The App Locally

### navigate to the front-end project

    ```
    cd churpy-dashboard
    ```

### Install dependencies

        ```
        $ yarn
        ```

### Run the app

        ```
        $ yarn start
        ```

### The app should be running on `localhost:3000`
